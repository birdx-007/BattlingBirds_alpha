# 《夺蛋录》alpha体验版

*Made by Bx.*

是本人的第一个比较大的**个人**项目(^^;;)，使用Qt5框架、C++语言编写的stg游戏。

目前已更新到体验版，拥有1个教程关及3个难度的3个关卡，4种不同的可高低切的机体。

相关展示视频及可游玩版下载地址请见[【全新发布】自制《愤怒的小鸟》同人STG——《夺蛋录(体验版)》](https://www.bilibili.com/video/BV1SS4y1w7q6)。
